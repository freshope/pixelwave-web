개인정보처리방침

시행일: 2026년 5월 19일  |  최종 수정: 2026년 5월 19일

오늘 하루(이하 "서비스")는 이용자의 개인정보를 소중히 여기며, 「개인정보 보호법」 및 관련 법령을 준수합니다. 본 방침은 서비스가 어떤 정보를 수집하고, 어떻게 사용하며, 어떻게 보호하는지 설명합니다.

1. 수집하는 개인정보 항목

서비스는 별도의 회원가입 없이 디바이스 식별자 기반으로 운영됩니다.

항목수집 방법목적
디바이스 식별자
(앱에서 생성한 고유 ID)		 앱 최초 실행 시 자동 생성 사용자 식별, 데이터 분리 보관
이름, 생년월일 이용자 직접 입력 (선택) 메인 화면 인사말 표시, 살아온 일수 계산
본인 휴대폰 번호 이용자 직접 입력 (선택) 위급 메시지 발송 시 발신자 식별 보조
비상 연락처
(가족·지인의 휴대폰 번호)		 이용자 직접 입력 (선택) 미클릭 누적 시 위급 메시지 자동 발송 대상
위급 메시지 본문 이용자 직접 입력 (선택) 비상 연락처로 발송할 메시지 내용
알림 설정
(주기 1·3·5일, 알림 시간, ON/OFF)		 이용자 직접 입력 로컬 알림 노출 및 위급 메시지 발송 주기 판단
생존 신고 기록
(클릭 일시)		 이용자가 버튼을 누른 시점에 자동 기록 살아온 일수 산정, 위급 메시지 발송 여부 판단
디바이스 푸시 토큰 OS 가 발급, 앱이 자동 전송 알림 발송 (선택적 기능)
광고 식별자(IDFA/AAID) OS 광고 SDK 가 제공 AdMob 배너 광고 노출 및 한도 제어

서비스는 위 항목 외의 민감정보(주민등록번호, 금융·건강 정보 등)를 수집하지 않습니다.

2. 비상 연락처 관련 안내

비상 연락처는 이용자가 직접 입력한 제3자(가족·지인)의 연락처입니다. 이용자는 해당 연락처의 본인으로부터 사전 동의를 받은 뒤 입력할 책임이 있습니다. 비상 연락처 정보는 위급 메시지 발송 외의 목적으로 사용되지 않으며, 광고·마케팅에 활용되지 않습니다.

3. 개인정보의 이용 목적

  • 매일 생존 신고 기능 및 살아온 일수 계산
  • 설정한 시간의 알림 노출
  • 지정한 주기(1·3·5일) 미클릭 시 비상 연락처로 위급 메시지 발송
  • 서비스 이용 통계 산출 및 오류 개선
  • 법령상 의무 이행

4. 개인정보 보유 및 이용 기간

이용자가 앱을 삭제하거나 설정 화면에서 데이터 삭제를 요청한 시점에 즉시 파기합니다. 위급 메시지 발송 이력은 발송 신뢰성 확인 및 분쟁 대응을 위해 발송일로부터 6개월간 보관 후 파기합니다. 관련 법령에 따라 일정 기간 보존이 요구되는 경우 해당 기간 동안만 보관 후 파기합니다.

5. 개인정보의 제3자 제공

서비스는 이용자의 개인정보를 원칙적으로 제3자에게 제공하지 않습니다. 다만, 다음의 경우는 예외로 합니다.

  • 이용자가 사전에 동의한 경우 (예: 비상 연락처 본인에게 위급 메시지 발송)
  • 법령의 규정에 의거하거나 수사 기관의 요청이 있는 경우

6. 개인정보 처리 위탁

서비스는 원활한 서비스 제공을 위해 아래와 같이 처리를 위탁합니다.

수탁 업체위탁 내용보유 기간
Supabase, Inc. 데이터베이스 저장 및 인증 인프라 운영 서비스 이용 기간
(주)솔라피 (SOLAPI) 위급 메시지 SMS 발송 게이트웨이 관련 법령에서 정한 기간
Google LLC 푸시 알림(FCM), AdMob 배너 광고 송출 각 사 정책에 따름
Apple Inc. iOS 푸시 알림(APNs) 각 사 정책에 따름
Cloudflare, Inc. 도메인·DNS·정적 페이지 호스팅·이메일 라우팅 서비스 이용 기간

7. 이용자의 권리와 행사 방법

이용자는 언제든지 다음의 권리를 행사할 수 있습니다.

  • 개인정보 열람 요청
  • 개인정보 수정·삭제 요청 (앱 설정 화면에서 직접 수정·삭제 가능)
  • 개인정보 처리 정지 요청
  • 앱 삭제를 통한 디바이스 식별자 기반 데이터 일괄 삭제 요청

권리 행사는 아래 문의처로 이메일 요청 시 지체 없이 처리합니다.

8. 개인정보의 안전성 확보 조치

  • 모든 데이터 전송은 HTTPS(TLS)로 암호화
  • Supabase Row Level Security(RLS)를 통한 디바이스별 데이터 격리
  • SMS 게이트웨이 호출은 서버 측에서만 수행, API 키는 서버 환경 변수로만 보관
  • 서버 접근 권한 최소화 및 주기적 점검

9. 광고 및 자동 수집 도구

서비스는 Google AdMob 을 통해 배너 광고를 표시합니다. 광고 송출 과정에서 OS 가 제공하는 광고 식별자(IDFA/AAID)와 일반적인 디바이스 정보(기기 모델, OS 버전 등)가 Google 로 전송될 수 있으며, 해당 정보의 처리는 Google 의 개인정보처리방침을 따릅니다. 이용자는 디바이스 OS 설정에서 광고 추적을 제한할 수 있습니다.

서비스는 알림 시간 관리를 위해 기기 로컬 저장소(localStorage / Preferences)를 사용하며, 별도의 행태 분석·마케팅 추적 도구는 사용하지 않습니다.

11. 방침 변경

본 방침이 변경될 경우 시행 7일 전에 앱 또는 본 페이지를 통해 공지합니다. 중대한 변경의 경우 30일 전에 공지하며, 필요한 경우 이용자의 동의를 별도로 받습니다.
